# Token流可视化工具

## 功能说明

本工具提供词法分析结果的可视化展示，包括：
- 彩色高亮不同类型的Token
- Token分类统计
- 直观的格式化输出

## 功能特点

1. **彩色显示**：
   - 关键字：蓝色
   - 标识符：绿色
   - 常数：黄色
   - 运算符：紫色
   - 界符：青色
   - 注释：红色

2. **详细信息**：
   - Token序号
   - Token类型
   - Token内容
   - Token编号

3. **统计功能**：
   - 总Token数
   - 各类型Token的数量

## 编译方法

```bash
cd visualization_tool
g++ -std=c++11 -o visualizer visualizer.cpp
```

## 使用方法

### 方式1：管道输入

```bash
echo "int main() { return 0; }" | ./visualizer
```

### 方式2：文件重定向

```bash
./visualizer < test.c
```

### 方式3：交互输入

```bash
./visualizer
# 输入代码，按Ctrl+D结束
```

## 技术实现

1. **ANSI颜色代码**：使用终端转义序列实现彩色输出
2. **Token分类**：根据编号自动分类Token类型
3. **格式化输出**：使用`iomanip`库实现对齐
