# 数字识别测试

## 功能说明

本测试程序专门测试词法分析器对各种数字格式的识别能力。

## 测试覆盖

1. **整数**：
   - 普通整数：`123`
   - 零：`0`
   - 大整数：`999999`

2. **浮点数**：
   - 标准浮点数：`3.14`
   - 小于1的小数：`0.5`

3. **科学计数法**：
   - 正指数：`1.5e10`
   - 负指数：`2E-5`
   - 显式正号：`3.14e+8`

4. **数字后缀**：
   - 长整数：`123L`
   - 无符号：`456U`
   - 浮点：`3.14F`
   - 组合：`100UL`

## 编译和运行

```bash
cd number_recognizer_test
g++ -std=c++11 -o number_test number_test.cpp
./number_test
```

## 技术要点

1. **DFA状态机**：数字识别使用多状态自动机
2. **向前看**：处理小数点和科学计数法需要lookahead
3. **后缀处理**：正确识别L、U、F等后缀
